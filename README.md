# Weather-App
This is a static server which gives You the current weather information of the city you select ..

to make it work on local host(on your pc):-
1) download these files in your pc
2) install node package in that file through terminal
3) install various packages we have required in the app.js file
4) and the end install nodemon package 
5) and made nodemon initialize by "nodemon init" (you should be present in the file where app.js is present)
6) and keep entering with by default options in next steps.
7) and type " nodemon app.js" and search localhost:3000 in your web browser.
8) BOOOOOOOOM!!!! your website is UP!!
